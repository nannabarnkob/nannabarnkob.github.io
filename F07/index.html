Sune's lecture

What if the data is so complex that there is no way to envision it?
D3 is not always good for exploratory analysis. How do people meet in the real world? No one has ever collected data on a big population before.
In the project, bluetoooth was a proxy for face-to-face meetings. Each encounter creates two data points which totaled to 7.362.231 links. In literature, this should result in a community structure of family, soccer teams, school classes etc.
This was claimed to be wrong - in the real world, one node is common for all making a big hair ball. Slicing by the hour and five minutes, this reveals certain clusters.
